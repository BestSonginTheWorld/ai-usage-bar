AIUsageMenuBar 설치 안내

배포 파일
- `AIUsageMenuBar-<version>.pkg`

설치 방법
1. GitHub Releases에서 최신 `.pkg` 파일을 다운로드합니다.
2. `AIUsageMenuBar-<version>.pkg`를 더블클릭합니다.
3. macOS 설치 마법사를 따라 설치합니다.
4. 설치가 끝나면 `/Applications/AIUsageMenuBar.app`를 한 번 직접 실행합니다.

설치 후 동작
- 앱 위치: `/Applications/AIUsageMenuBar.app`
- 설정/로그/캐시: `~/Library/Application Support/AIUsageMenuBar/`
- 자동 시작 설정 파일: `~/Library/LaunchAgents/local.ai-usage-menubar-app.plist`
- 앱을 한 번 실행하면 필요한 collector/runtime 파일과 자동 시작 설정이 생성됩니다.
- 자동 시작은 다음 로그인부터 적용됩니다.

필수 환경
- macOS 13 이상
- `tmux`
- `claude` CLI
- `codex` CLI

필수 도구가 없을 때
- 앱 설치와 실행 자체는 됩니다.
- 하지만 사용량 수집은 동작하지 않습니다.
- 메뉴에서 provider가 `unavailable` 또는 `not installed` 상태로 보일 수 있습니다.

처음 실행 시 macOS 경고가 뜰 수 있음
- 이 앱은 아직 Apple notarization이 되어 있지 않습니다.
- 다른 Mac에서는 처음 실행할 때 보안 경고가 뜰 수 있습니다.
- 이 경우 Finder에서 `AIUsageMenuBar.app`를 우클릭한 뒤 `열기`를 선택하거나, 시스템 설정 > 개인정보 보호 및 보안에서 `그래도 열기`를 선택하면 됩니다.

삭제 방법
1. 앱을 종료합니다.
2. `/Applications/AIUsageMenuBar.app`를 삭제합니다.
3. 필요하면 아래 경로도 같이 삭제합니다.

   `~/Library/LaunchAgents/local.ai-usage-menubar-app.plist`
   `~/Library/Application Support/AIUsageMenuBar/`

개발자용 수동 제거 스크립트
- 저장소에는 `scripts/uninstall.sh`가 포함되어 있습니다.
- 이 스크립트는 `/Applications/AIUsageMenuBar.app`, `~/Applications/AIUsageMenuBar.app`, LaunchAgent를 함께 정리합니다.
